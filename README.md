### 视频里曾经使用过的地震信息滚动条/字幕，因为NQserver api不再对外开放，因此开源
# 本字幕特点：
1.支持在接收到新情报时滚动条 **不会打断重跑！**

2.支持 **显示JMA的警报区域或予报时显示震度3/4以上的预想区域**（需要jma有发表）

3.超快的翻译：在收到ShakeAlert情报时 **1s内翻译完成！**

4.**超多情报支持显示** ：ShakeAlert eew，中国地震局（CEA）eew（震级3.0及以上），福建地震局eew，四川地震局eew，成都高新减灾所（ICL）eew，中央气象署（CWA）EEW，日本气象厅（JMA）EEW，日本气象厅（JMA）地震情报，中央气象署（CWA）地震情报，中国地震台网（CENC）测定信息

相关py文件用处的注释已放在**每个py文件的第一行**，因此不在此处说明用处
